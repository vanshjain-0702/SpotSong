# SpotSong
SpotSong is a modern music streaming web application prototype built with a sleek dark theme and vibrant yellow highlights. It delivers a smooth, engaging user experience while showcasing the essential features of a streaming platform.
Features
- Playlist Management: Create, view, and organize playlists with custom styling.
- Music Controls: Play, pause, next, previous, shuffle, like, and volume options for seamless playback.
- Navigation Sidebar: Quick access to Browse, Radio, Your Library, Liked Songs, and Playlist creation.
- Discover Section: Suggested songs, podcasts, and artist highlights to keep users engaged.
- Search Functionality: Responsive search bar for finding songs and artists
